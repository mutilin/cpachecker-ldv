CPAchecker Installation
=======================
If you have downloaded the CPAchecker source,
see BUILD.txt for information on how to build CPAchecker.

1. You need a Java Runtime Environment which is at least Java 6 compatible
   (e.g., Sun/Oracle JRE, OpenJDK).
   See http://java.oracle.com/ or install a package from your distribution.
2. Install GNU Multiprecision Arithmetic Library libgmp.so.3 and libgmpxx.so.4.
   (Ubuntu: sudo apt-get install libgmp3c2 libgmpxx4ldbl)
3. Extract the content of the CPAchecker zip file into a directory of your choice.

For information on how to run CPAchecker, see README.txt.
